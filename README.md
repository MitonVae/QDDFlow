# QDD Flow — 游戏任务流程可视化编辑器

> 一个面向游戏策划的轻量级 Web 工具，用于快速搭建、可视化和导出游戏任务/关卡流程。  
> 纯前端，无需服务器，数据存储在浏览器本地（localStorage + IndexedDB）。

---

## 📸 核心功能一览

### 🏠 首页 — 多 QDD 管理

- 支持创建多个 QDD（Quest Design Document，任务设计文档）
- 每个 QDD 独立管理，可**重命名**、**删除**、**分享**
- 刷新页面自动恢复上次打开的视图

### 📋 编辑器 — 环节设计

进入某个 QDD 后，进入编辑模式：

| 功能 | 说明 |
|------|------|
| **环节列表**（左侧面板） | 添加/删除/拖拽排序环节，支持批量操作 |
| **可视化预览**（中央画布） | 实时预览流程图，支持直接在预览区编辑内容 |
| **属性面板**（右侧弹出） | 点击环节标题打开，编辑详细属性（触发方式、地点、人物、描述等） |

### 🎨 两种布局模式

| 布局 | 特点 |
|------|------|
| **📋 表格式** | 每个环节为一列，所有字段按行展示，适合详细信息一览 |
| **🎯 时间轴式** | 横向时间线，突出流程顺序，适合演示和审查 |

通过工具栏的「布局」下拉菜单切换，支持实时调整**列宽**（120px ~ 380px）。

## 🎮 任务类型系统

每个环节可标记一种任务类型，类型决定环节标题的颜色标识：

| 类型代码 | 名称 | 颜色 | 适用场景 |
|----------|------|------|----------|
| `PUZ` | 玩法 | 🟣 紫色 `#7c6af7` | 解谜、潜行、互动等玩法环节 |
| `BAT` | 战斗 | 🟢 绿色 `#06d6a0` | 战斗环节 |
| `NAV` | 跑图 | 🟡 黄色 `#ffd166` | 纯移动、探索地图 |
| `TL` | TL 剧情 | ⬛ 深灰 `#6b7280` | Timeline 过场动画、走路演出 |
| `PHONE` | 电话 | 🔘 浅灰 `#b0b8c4` | 角色通话形式的剧情 |
| `SMS` | 短信 | 🔘 浅灰 `#b0b8c4` | 文字消息形式的剧情 |
| `DIA` | 对话 | 🔘 浅灰 `#b0b8c4` | 面对面 NPC 对话 |

> 💡 在预览区点击环节标题旁的类型徽标即可快速切换。还支持**自定义类型**（自动持久化）。

## 🔧 触发方式

每个环节可设定触发条件，内置选项：

- `Room触发` — 玩家进入区域自动触发
- `接续自动触发` — 上一环节结束后自动衔接
- `剧情触发` — 特定剧情条件达成后触发
- `玩家选择` — 需要玩家主动操作
- `接续` — 简单顺序接续

> 同样支持**自定义触发方式**。

---

## 📝 环节属性

每个环节支持以下字段（均可在属性面板或预览区直接编辑）：

| 字段 | 说明 |
|------|------|
| **名称** | 环节标题，支持序号前缀（如 `0.任务接取`） |
| **任务类型** | 见上表 |
| **触发方式** | 该环节如何触发 |
| **地点** | 场景名称 |
| **出场人物** | 涉及的 NPC/角色 |
| **描述** | 详细设计说明 |
| **配图** | 支持拖拽上传、点击上传、粘贴图片 |
| **自定义字段** | 任意 key-value 对，如"玩法类型"、"敌人"、"奖励"等 |

---

## 🔢 序号自动重排

如果环节标题以数字开头（如 `0.任务接取`、`1.大厅探索`），当你**拖拽排序**或**移动环节**后，序号会自动按新顺序重新编号：

```
拖拽前：0.接取 → 1.探索 → 2.伪装 → 3.战斗
把"2.伪装"拖到最后：
拖拽后：0.接取 → 1.探索 → 2.战斗 → 3.伪装  ← 自动修正
```

- 保留原始分隔符（`.`、`、`、`-`、空格）
- 仅当 ≥2 个环节有序号前缀时才启用
- 没有序号前缀的标题不受影响

## 🤖 AI 导入 / 导出

### AI 导入

1. 点击工具栏 **「🤖 AI 导入」**
2. 复制左侧的提示词（Prompt），发给任意 AI（ChatGPT、Claude 等）
3. 把 AI 输出的 JSON 粘贴到右侧文本框
4. 选择导入模式：
   - **追加** — 在现有环节后面追加
   - **修改** — 按名称/编号匹配，更新已有环节
   - **新建** — 创建为全新的 QDD
5. 点击「导入」

> 也支持从 `.json` 文件导入（含图片的完整备份）。

### AI 导出

- **复制文字 JSON** — 不含图片，直接发给 AI 让它续写/修改
- **下载完整备份** — 含图片的 `.json` 文件，可随时还原

## 📋 标题链复制

点击工具栏 **「📋 标题链」** 按钮，一键复制所有环节标题，用箭头连接：

```
0.任务接取 → 1.武康大楼 → 2.对话门卫 → 3.伪装电工 → 4.楼道探索
```

适合粘贴到文档或聊天中快速分享流程概要。

## 📤 导出功能

| 功能 | 说明 |
|------|------|
| **🖼️ 导出 PNG** | 将预览区渲染为高清 PNG 图片（2x 缩放） |
| **📄 导出 PDF** | 将预览区渲染为 PDF 文档 |
| **🔗 分享** | 生成包含数据的分享链接（适合轻量内容） |

> 导出时，没有图片的环节会自动隐藏图片占位框，保持输出整洁。

## 📂 导入表格

支持从 **Excel (.xlsx/.xls)** 或 **CSV** 文件导入环节：

1. 点击工具栏 **「📂 导入表格」**
2. 选择文件后弹出列映射对话框
3. 将文件中的列与 QDD 字段对应（自动识别）
4. 点击「导入」

## 🖼️ 图片管理

- **上传方式**：点击占位区、拖拽文件、Ctrl+V 粘贴
- **存储**：图片自动存入浏览器 IndexedDB，不占 localStorage 配额
- **预览**：可放大查看、替换、删除

---

## 💾 数据存储

| 存储位置 | 内容 |
|----------|------|
| `localStorage` | QDD 结构数据、用户偏好（主题/布局/列宽）、自定义类型 |
| `sessionStorage` | 当前会话的完整备份（防止 localStorage 配额裁剪丢数据） |
| `IndexedDB` | 所有上传的图片（base64） |

- 自动保存：编辑后自动保存，工具栏显示最后保存时间
- 撤回/重做：支持 `Ctrl+Z` 撤回、`Ctrl+Y` 重做

---

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|

## 🚀 使用方式

1. 用浏览器直接打开 `index.html`（推荐 Chrome / Edge）
2. 无需安装任何依赖，无需服务器
3. 所有数据存储在浏览器本地

> ⚠️ 清除浏览器数据会丢失所有 QDD，建议定期使用「AI 导出 → 下载完整备份」。
